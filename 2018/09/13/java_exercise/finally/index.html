  • finally 块会执行吗?

执行结果

finally 块执行了

在 Eclipse 断点调试，发现 finally 确实执行，查看 Oracle 的官方文档，解释了 finally 只有在 JVM 退出了或进程被 kill 掉了才不会执行。

返回结果是 10，为什么?

在 Eclipse 断点调试，发现 finally 确实执行了 i = i + 10, i 的值被修改为 20.但是最后栈顶的值设置为10，返回的还是 10.
